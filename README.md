#LPTours Web service

## Endpoint
  
  ```sh
 http://lptours.herokuapp.com/
 ``` 
## Description

This is the web service of the API [ZMZ_Tours] (https://github.com/ZhongMeiZhou/scraper_webAPI)

## Usage and Examples
Handles:

- GET /
  - ZMZ Welcome page

- GET /tours
  - Explore Tours - search of tours by a given country

- GET /tours/param[:id]
  - Country tours with known number ID
